Data Science Portfolio

A collection of recent projects.

  • SugarTime

    The SugarTime package is a set of tools to model the effects of carbohydrates and insulin on blood glucose dynamics. The goal of these tools is to learn how a diabetic's blood glucose responds to food and to insulin. After it has learned this, the patient can provide the timing and amount of carbs they plan on eating, and the model will offer a best guess as to which insulin dosage will keep their blood glucose within an optimal range.

  • SearchMo

    This started off as an exploration of my Mormon roots. It is currently an app that provides N-Gram frequency search functionality for a corpus of sermons and lessons from LDS church leadership. It covers from the mid 1800s to today. I have plans to expand into other religions/traditions.

  • DuraScrape

    DuraScrape is a web crawler that scrapes open-access journal articles from academic journal websites and saves them in a SQL database.

  • Countdown

    This is currently a quick EDA of the mortality dataset at NBER. The short-term goal is to provide a clean interface to the dataset, allowing users to input demographic features (e.g., race, education, sex, marital status, etc.), and then see when someone with those demographic features is most likely to die.

  • NERF: Nonlinear Estimation of Receptive Fields

    This is currently a repository of MATLAB and python code for preprocessing and analysis of behavioral and neural timeseries data. Beyond I/O, cleaning, and basic plotting functions, it specifically provides tools for inferring the spatiotemporal preferences of visually responsive neurons.
