Data Science Portfolio

A collection of recent projects.

  • The language of faith

    This started off as an exploration of my Mormon roots. It is currently an app that provides N-Gram frequency search functionality for a corpus of sermons and lessons from LDS church leadership. It covers from the mid 1800s to today. I have plans to expand into other religions/traditions.

  • Countdown

    This is currently an EDA of the mortality dataset at NBER. The short-term goal is to provide a clean interface to the dataset, allowing users to input demographic features (e.g., race, education, sex, marital status, etc.), and then see when someone with those demographic features is most likely to die.

  • NERF: Nonlinear Estimation of Receptive Fields

    This is currently a repository of code for preprocessing and analysis of behavioral and neural timeseries data. Beyond I/O, cleaning, and basic plotting functions, it specifically provides tools for inferring the spatiotemporal preferences of visually responsive neurons.
